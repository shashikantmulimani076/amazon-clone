
# Amazon Clone 🛒

A responsive Amazon homepage clone built using HTML and CSS.
This project helped me practice frontend development concepts like Flexbox, Grid, positioning, animations, media queries, and responsive layouts.
link:-https://shashikantmulimani076.github.io/amazon-clone/
## 🚀 Features

* Amazon-style navigation bar
* Hero section with banner image
* Product cards layout
* Responsive design
* Footer section
* Hover effects
* CSS styling similar to Amazon UI

## 🛠️ Technologies Used

* HTML5
* CSS3

## 📚 What I Learned

* Flexbox
* CSS Grid
* Positioning
* Media Queries
* Hover Effects
* Responsive Web Design
* CSS Animations
* Box Shadow & Filters
